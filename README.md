# Задание 5.1 "Новые типы"
![CI](https://github.com/itsOleggio/AdvancedJS_T5_1/actions/workflows/ci.yml/badge.svg)

## Описание 

Создание Typescript модуля, содержащего классы **Product** и **Film**. 
Класс **Film** наследуется от **Product**. Содержит в себе собсвенные поля:
- **duration**: продолжительность
- **slogan**: слоган

Остальные поля берутся от родителя **Product**: 

- **productName**: название
- **releaseYear**: год выпуска
- **genre**: жанры

Также добавлены интерфейсы **IProduct** и **IFilm** для общего использования в классах.

## Тесты

Используем библиотеку Jest.
Тестом покрыта основная функция программы `getInfo()`

Запуск тестов: `npx test`. 

## Код программы

Программа находится в папке `src`.
Тесты ёт в папке `/src/ts/__tests__/`.

## Запуск программы

Сборка программы: `npm build`
Запуск программы: `npm start`.
